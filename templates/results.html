Algorithm Explanation
{% if algorithm == "Theta*" %}

Theta* is an any-angle pathfinding algorithm that extends A* by allowing paths to go through any angle, not just along grid lines.

Key features:

  • Uses line-of-sight checks to create smoother paths
  • Reduces unnecessary waypoints
  • Creates more natural-looking paths

Key features:

  • Guarantees the shortest path when using an admissible heuristic
  • Efficiently explores the search space
  • Works well for grid-based movement
