About the Algorithms

A* is a popular pathfinding algorithm that uses a heuristic to guide the search towards the goal. It guarantees the shortest path when using an admissible heuristic.

In this implementation, A* uses a grid-based movement with 4 directions (up, down, left, right) and Euclidean distance as the heuristic.

Theta* is an extension of A* that allows for any-angle paths. It checks for line-of-sight between non-adjacent nodes, allowing for smoother and more natural paths.

This implementation uses line-of-sight checks to connect nodes that are not adjacent but have a clear path between them.

After finding a path, the algorithm can optimize it by removing unnecessary waypoints when a direct line-of-sight exists.

The optimization process includes:

  • Forward optimization: Checking for direct paths from each waypoint to future waypoints
  • Reverse optimization: Checking for direct paths from the goal back to earlier waypoints
  • Segment splitting: Breaking long segments into smaller ones for better optimization
